# DIGITAL_IMAGE_STEGANOGRAPHY
A C-based steganography project that hides and retrieves secret messages within image files using file handling and bitwise operations.

# Steganography in C

This project demonstrates how to hide and extract secret messages within image files using C language.

## Features
- Encode text messages into image files
- Decode and retrieve hidden messages
- File handling and bitwise manipulation
- Command-line interface for user input

## How to Run
1. Compile the project using any C compiler:
   gcc -o stego main.c encode.c decode.c   //-o will create your own executable file
2.Run the program:
./stego
